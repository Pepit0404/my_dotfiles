# my_dotfiles

Simple program for initializing my application settings

# Installation

## Clone the project

_**The project must be cloned in the Home directory!**_

```bash
git clone git@github.com:Pepit0404/my_dotfiles.git
cd my_dotfiles
```

_**DO NOT CHANGE THE FOLDER NAME !**_

## Commande

```bash
sudo ./scripts/init.sh
./scripts/setup.sh
```

## Set zsh as default

```bash
chsh -s $(which zsh)
```

Then restart your pc
